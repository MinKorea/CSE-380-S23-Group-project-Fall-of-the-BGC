Fall of the BGC
Game Design Document

by Min Kwak, Alexander Gentile, Joyce Qiu

INTRODUCTION

This document describes a game called "Fall of the BGC" which is designed to be a top-down roguelike game that will use custom music, tilemaps, enemy, and player sprites. You have to challenge defeating enemies and the final boss. Have fun!


TECHNOLOGY

Fall of the BGC will be made using a game engine called Wolfie2D which uses Typescript. Tilemaps and Tiles will be created using Piskel and Tiled.


BACK STORY

The Bad Guys Corporation(BGC) has done lots of very very bad things whatever you imagine. And the boss overworks his employees every day. The Smith bros and employees overworked every day got suddenly kicked out from the BGC. So, the family requests the Good Guys Corporation (GGC) to get their money back. The Player, whose name is Good Guy Lukas, has been given a mission by the GGC to take down the BGC.

GAMEPLAY

The player will begin their mission on the first floor of the building, their starting weapon is just their fists which they can use to attack enemies. On each floor there are rooms and these rooms are categorized into 3 types. There are enemy rooms, shop rooms, and boss rooms.

  • Enemy rooms: There are 3 enemy rooms per floor. These rooms consist of enemies that want to attack you. On the first floor there will be 3-5 enemies, second floor 5-7, third floor 7-9.
  • Shop rooms: These rooms are a safe space where you can purchase new weapons and abilities using money that the enemies drop when you defeat them.

  • Boss rooms: These rooms are where you can fight the boss on the floor. Beating the boss gives you access to go to the next floor.

CONTROLS

This game will be played using both a keyboard and mouse. Once started, use the following:

  • W - Move Up
  • S - Move Down
  • A - Move Left
  • D - Move Right
  • SPACE - Attack
  • M - Shows map of the floor
  • ESC - This pauses the game and shows the paused menu

GRAPHICAL USER INTERFACE

As far as the GUI is concerned, if we consider

  • Splash Screen - The Splash Screen GUI will consist of the game logo as the background of the startup screen with Text that says "Click to Start".

  • Main Menu - The Main menu GUI will show after the Splash Screen is clicked. The main menu will have 4 buttons:
    • Start - Pressing this button will start the game
    • Controls - Pressing this button will move to the Controls Screen GUI
    • Display Controls - If pressed, the game will display an info screen that includes a description of all game controls.
    • Help - Pressing this button will change the viewport to the Help Screen GUI
    • Level Selection - Pressing this button will change the viewport to the Level Selection Screen GUI

  • Level Selection Screen - The Level Selection Screen GUI will have 3 options:
    • Floor 1 - Choosing this option brings you to level 1
    • Floor 2 - Choosing this option brings you to level 2
    • Floor 3 - Choosing this option brings you to level 3
  • Controls Screen - The Controls Screen GUI will show the controls of the game and also has a back-to-the-main menu option that returns to the main menu screen.
  • Help Screen - The Help Screen GUI gives a description of the backstory of the game, the name of the developers, and the cheat codes that can be used. There is also a back-to-the-main menu option that returns to the main menu screen.
    • Cheat codes - gotolvl1(player teleports to first level), gotolvl2(player teleports to the second level), gotolvl3(player teleports to third level), maxmoney(player gains the max amount of money), godmode(player can no longer die).
  • In-Game Screen - The In-Game Screen will display the player inside a room that will fill the entire game port. On each edge of the room, there is a door that will bring you to another room. At the top left of the screen, there is a health bar display that shows the player's current health points. Also, at the top right of the screen, there is a pause icon that can be clicked to pause the game, you can also click ESC to pause the game as well.
  • Paused Screen - The Pause Screen GUI has 4 options:
    • Resume - This option resumes the game
    • Controls Screen - This option display the Controls screen
    • Help Screen - This option displays the Help Screen
    • Return to the Main Menu - This option returns you to the main menu screen

    • Player/Enemy/Boss Walking
    • Player/Enemy/Boss taking damage
